# Скачать tar.xz
https://desktop.telegram.org/

# Переместить в /opt/
```bash
sudo mv tsetup...tar.xz /opt/
cd /opt
```

# Разарзивировать
```bash
sudo tar -xvf tsetup...tar.xz
```

# Установить телеграм
```bash
cd Telegram
./Telegram
```
